My Minecraft servers

I currently have one public Minecraft server, it's still a BETA version and it's available for 1.18. The server software is Fabric but you can join with Vanilla without any problem, the theme is a survival server without hacks or anything that may alter the survival experience. Due security reasons for the players, the server is only available for players who use the official Minecraft Launcher, so cracked players cannot join.
The server will be manually started everytime someone wants to join, so you can contact me to start it or even give you access to start the server if you have an Aternos account. There's currently no Discord server for it so you can find me with the tag Alex22#7756.
