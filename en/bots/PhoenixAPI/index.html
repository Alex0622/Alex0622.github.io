PhoenixAPI

PhoenixAPI is a bot developed with discord.js. The main reason of this bot is the managment of exaroton servers using the exaroton API. You can find the documentation about the exaroton API here and its package here.

About PhoenixAPI

Because exaroton requires a private API token to get access to your servers and account information the bot cannot be public, so we have developed a repository that you can download and set up to manage your exaroton servers.

Check out the source on GitHub , check out our Spanish intallation on or join our Discord server for more information and support.

PhoenixAPI's commands

account, ban, deop, dynip, execute, help, info, list, log, op, pardon, ram, restart, servers, start, status, stop, unwhitelist, whitelist.
