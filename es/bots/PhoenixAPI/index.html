PhoenixAPI

PhoenixAPI es un bot de Discord desarrollado con discord.js. El objetivo principal del bot es la administración de servidores de exaroton usando la API de exaroton.

Acerca de PhoenixAPI

Ya que exaroton requiere un token para la API para poder acceder a tu cuenta y servidores de exaroton, el bot no puede ser público; hemos desarrollado este repositorio que puedes descargar y configurar para administrar tus servidores de exaroton.

Encuentra la fuente en GitHub , revisa el tutoturial de instalación en YT o únete a nuestro servidor de Discord para más información o soporte.

Comandos de PhoenixAPI

account, ban, deop, dynip, execute, help, info, list, log, op, pardon, ram, restart, servers, start, status, stop, unwhitelist, whitelist.
